# SmrtDoodle — Microsoft Store Listing

## Short Description (max 100 chars)
A powerful image editor with layers, blend modes, AI tools, and PSD support for Windows.

## Full Description

SmrtDoodle is a full-featured image editor designed for Windows, built with modern WinUI 3 and hardware-accelerated Win2D rendering.

### Features

**Drawing Tools**
- Pencil, Brush (8 styles including Calligraphy, Airbrush, Oil, Watercolor), and Eraser
- Gradient, Blur, Sharpen, Smudge, Clone Stamp, and Pattern Fill tools
- Line, Curve, and Shape tools with 12 shapes and 3 fill modes
- Flood Fill with adjustable tolerance
- Measure tool with distance and angle readout
- Eyedropper/Color Picker

**Layer System**
- Unlimited layers with opacity, visibility, and lock controls
- 25 Photoshop-compatible blend modes (Normal, Multiply, Screen, Overlay, and more)
- Layer groups, effects (Drop Shadow, Inner Shadow, Glow, Stroke), and masks
- Adjustment layers (Brightness/Contrast, Hue/Saturation, Color Balance, Levels, Curves)
- Inline rename, drag reorder, duplicate, merge, and flatten

**File Format Support**
- Native .sdd project format (preserves all layers)
- Import/Export: PNG, JPEG, BMP, GIF, PSD, PSDT, TIFF, WebP, ICO, SVG, TGA, DDS, PDF
- PSD import with layer preservation

**Selection Tools**
- Rectangle and Free-form Lasso selection
- Floating selection with move support
- Transparent selection mode

**Image Transforms**
- Resize, Crop, Flip (H/V), Rotate (90/180/270), Skew
- Invert Colors

**AI-Powered Tools (Pro)**
- Background Removal
- Image Upscaling (2x/4x Super-Resolution)
- Content-Aware Fill
- Auto-Colorize
- Style Transfer
- Smart Selection
- Noise Reduction

**Canvas & View**
- Zoom 10–800% with smooth scaling
- Grid overlay and ruler guides
- Checkerboard transparency display

**Accessibility**
- Full keyboard navigation with access keys for all tools
- Screen reader support with ARIA landmarks and live regions
- High contrast theme compatible
- RTL layout support (Arabic, Hebrew, Urdu, Farsi)

**Localization**
- English, Spanish, German, French, Japanese, Chinese (Simplified), Arabic, Urdu, Portuguese (Brazil)

**Integration**
- SmrtPad integration for embedded drawings
- Drag & drop file support
- Print with system print dialog
- Clipboard copy/paste

### System Requirements
- Windows 10 version 1809 (build 17763) or later
- Windows 11 recommended
- DirectX 11 compatible GPU
- 4 GB RAM minimum, 8 GB recommended
